Insta Scraper

This bot is created for a person who is a Food Blogger. The Food Blogger wants to follow a lot of people so that he can get noticed. He also wants to do some other tasks. This bot helps the food blogger in completing these tasks. This bot uses automation using Selenium. 
1)	With this bot we can login to our Instagram handle using Selenium.
2)	It will type “food” in search bar and print all the names of the Instagram Handles that are displayed in list after typing “food”.
3)	Searching and opening a profile: It will search for a profile “So Delhi” and open it. 
4)	Follow/Unfollow Instagram handle: It will follow the handle “So Delhi” and display a message that we are already following it. After following, it will unfollow the Instagram handle and print a message accordingly.
5)	Like/Unlike posts: It will like the top 30 posts of “dilsefoodie”. It will print message if you have already liked it. It will unlike the top 30 posts of the “dilsefoodie”. It will print message if you have already unliked it.
6)	Extract list of followers: It will extract the usernames of the first 500 followers of ‘foodtalkindia’ and ‘sodelhi’. It will print all the followers of “foodtalkindia” that you are following but those who don’t follow our handle.
7)	It will check the story of ‘coding.ninjas’.
